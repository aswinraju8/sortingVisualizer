### ğŸ¨ How It Works

* A list of random integers is generated.
* Each integer is represented as a vertical bar whose height corresponds to its value.
* During sorting:

  * **Green** bars indicate elements being compared.
  * **Red** bars indicate elements being swapped.
* The screen updates dynamically to show progress through each iteration of the sorting algorithm.
